About Bible Reader

I created a Bible reader for Android. It is an open source project, under the Apache license.

The app is simple and easy-to-use, with minimalist design. It supports offline reading, multi-versions, full-text search, and many other features, while maintaining a small download and install size.

You can download the it from here. Recent changes are listed here. Feel free to request new features or report bugs here.

Main Features

  • Download it for free, and read the Bible ads-free
  • Read without network connection
  • Small download and installation size
  • Read multiple versions at the same time
  • Full-text search
  • Add bookmarks and highlights to your favorite verses
  • Add your own notes
  • Study with Strong’s Numbers
  • Share verses with friends through your favorite social media, email, SMS, etc.
  • Customize font size
  • Day / night mode

We currently support tens of different translations, with more added regularly. A full list of available translations can be found here.
